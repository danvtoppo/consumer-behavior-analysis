# Consumer Behavior Analysis: Retail Insights Project

### Project Overview
This project analyzes retail consumer behavior to identify spending patterns across different demographics. I used **DBeaver** and **PostgreSQL (Neon)** for data storage and analysis. 

### Stack
* **Database:** Neon (Cloud PostgreSQL)
* **SQL Editor:** DBeaver, Neon
* **Data Source:** Kaggle Retail Consumer Behavior Dataset

### Business Questions Answered
1. **Income vs. Spending:** Do high-income customers spend more, or are middle-income discount users more valuable?
2. **Impulse vs. Planned:** Are smartphone users significantly more likely to make impulsive purchases?
3. **Research vs. Returns:** Does spending more time on research lead to higher ratings and fewer returns?
